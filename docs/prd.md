# Seoul Craft Moment — Product Requirements Document (PRD)

> 이 문서는 **Claude Design**과 **Claude Code**가 공통으로 참조하는 기준 문서입니다.
> 디자인 방향, 페이지 구조, 섹션별 콘텐츠, 기술 범위는 모두 이 PRD를 따릅니다.

---

## 1. Project Overview

**Project Name:** Seoul Craft Moment

**Page Type:** React 기반 단일 페이지(One-page) 영어 랜딩페이지

**Core Message:** *A Quiet Korean Craft Experience in Seoul*

Seoul Craft Moment는 방한 외국인이 서울에서 한국의 공예 재료와 "손의 시간"을 조용하고 고급스럽게 경험할 수 있는 **프리미엄 공예 체험 서비스**입니다. 한지, 도자, 자개, 매듭, 섬유, 차, 향 등 한국적인 재료와 감각을 바탕으로 소규모 공예 체험을 제공합니다.

이 랜딩페이지의 역할은 서비스를 시각적으로 소개하고, 사용자를 향후 예약 또는 제품 탐색으로 자연스럽게 안내하는 것입니다.

---

## 2. Target Audience

- 방한 외국인 (서울 방문 여행자)
- 한국적인 로컬 체험을 찾는 여행자
- 조용하고 고급스러운 문화 경험을 원하는 개인, 커플, 소규모 그룹
- 원데이 클래스보다 깊이 있고 프리미엄한 경험을 선호하는 사용자

---

## 3. Service Concept

Seoul Craft Moment는 단순한 원데이 클래스가 아닙니다. 각 세션은 **한국 공예와의 조용한 만남**으로 설계됩니다 — 재료를 만지고, 그 이야기를 배우고, 손으로 만들고, 서울의 한 조각을 집으로 가져가는 경험입니다.

핵심 재료와 감각:
- 한지 (Hanji / Korean paper)
- 도자 (Ceramic / clay)
- 자개 (Mother-of-pearl)
- 매듭 (Knot)
- 섬유 (Textile)
- 차 (Tea)
- 향 (Scent)

이 서비스는 다음 단계로 확장될 수 있는 비즈니스입니다:
체험 예약 → 직접 제작 → 완성품 가져가기 → 큐레이션 공예품 구매 → 프라이빗 그룹 세션.

---

## 4. Business Flow

1. 공예 체험 예약 (Reserve a craft experience)
2. 손으로 공예품 제작 (Make a handmade object)
3. 서울의 기억으로 완성품 가져가기 (Take it home as a memory from Seoul)
4. 큐레이션 공예품 탐색 (Explore curated handmade objects)
5. 프라이빗 그룹 세션 요청 (Request a private group session)
6. 향후 온라인 제품 판매 또는 커스텀 기프트 주문으로 확장 (Later expand into online product sales or custom gift orders)

---

## 5. Page Goals

- 방한 외국인에게 **프리미엄 한국 공예 체험** 서비스임을 즉시 전달
- 조용하고 고급스러운, 한국적이지만 관광지스럽지 않은 무드 형성
- 서비스 콘셉트와 공예 프로그램을 명확하게 소개
- "만들고 → 가져가고 → 큐레이션 제품으로 이어지는" 비즈니스 흐름 시각화
- 사용자를 예약(Reserve) 및 제품 탐색(Explore)으로 자연스럽게 유도
- 프라이빗 그룹 세션 문의 유도

## 8. Design Direction

디자인은 다음과 같은 느낌이어야 합니다:

- Premium but not flashy (고급스럽지만 화려하지 않은)
- Korean but not touristy (한국적이지만 관광지스럽지 않은)
- Calm but not boring (차분하지만 지루하지 않은)
- Craft-based but modern (공예 기반이지만 현대적인)
- Warm, refined, and trustworthy (따뜻하고, 정제되고, 신뢰감 있는)
- Suitable for foreign visitors (외국인 방문자에게 적합한)
- High-end but approachable (고급스럽지만 다가가기 쉬운)

**Visual Mood:**

- Hanji texture (한지 질감)
- Ceramic white (도자기 화이트)
- Warm wood (따뜻한 우드톤)
- Soft shadow (부드러운 그림자)
- Thin lines (얇은 선)
- Quiet table (조용한 테이블)
- Museum shop premium (뮤지엄 숍 프리미엄)
- Calm Korean craft mood (차분한 한국 공예 무드)

---

## 9. Avoid List

다음 요소는 **사용하지 않습니다**:

- Purple (보라색)
- SaaS startup style (SaaS 스타트업 스타일)
- App promotion page style (앱 홍보 페이지 스타일)
- Neon colors (네온 컬러)
- Strong gradients (강한 그라데이션)
- Cold tech blue (차가운 테크 블루)
- Strong red/gold traditional tourist look (강한 레드/골드 전통 관광 스타일)
- Hanbok, palace, Taegeuk patterns (한복, 궁궐, 태극 문양)
- Folk village mood (민속촌 무드)
- Too many icons (과도한 아이콘)
- Cute hobby-class style (귀여운 취미 클래스 스타일)
- Cheap tour-product feeling (저렴한 관광 상품 느낌)

## 11. Technical Scope

**사용 기술:**

- React
- Vite
- One-page landing page
- Responsive web design (반응형 웹)

**구현하지 않는 항목:**

- Login (로그인)
- Database (데이터베이스)
- Payment (결제)
- Real booking system (실제 예약 시스템)
- Shopping cart (장바구니)
- User account (사용자 계정)
- Admin page (관리자 페이지)

랜딩페이지는 서비스를 **시각적으로 소개**하고, 사용자를 향후 예약 또는 제품 탐색으로 안내하는 역할만 수행합니다. CTA 버튼은 시각적 안내 요소로 동작하며 실제 백엔드 연동은 없습니다.

## 12. Future Expansion

향후 비즈니스 확장 방향:

1. 온라인 공예품 판매 (Curated Handmade Shop → 실제 커머스)
2. 커스텀 기프트 주문 (Custom gift orders)
3. 실제 예약 시스템 연동 (Reservation / booking system)
4. 프라이빗 그룹 세션 문의 및 관리 기능
5. 다국어 지원 확장 가능성
6. 메이커/아티스트 큐레이션 확대

> 현재 랜딩페이지에서는 위 기능들을 **비즈니스 확장 섹션**으로 시각적으로만 노출하며, 실제 기능은 구현하지 않습니다.

## 13. Success Criteria

- [ ] 방한 외국인이 3초 안에 "서울에서의 프리미엄 한국 공예 체험" 서비스임을 이해할 수 있다.
- [ ] 9개 섹션이 PRD 순서대로 모두 구현되어 있다.
- [ ] Hero의 핵심 메시지와 CTA(Reserve / Explore)가 명확하게 노출된다.
- [ ] 디자인이 Design Direction과 Visual Mood를 충족하고 Avoid List를 위반하지 않는다.
- [ ] 조용하고 고급스러운, 뮤지엄 숍 프리미엄 무드가 전달된다.
- [ ] 데스크톱과 모바일에서 반응형으로 자연스럽게 동작한다.
- [ ] Curated Shop / Private Sessions가 비즈니스 확장 섹션으로 자연스럽게 표현된다 (실제 기능 없이).
- [ ] 전체 톤이 관광지스럽지 않고, 한국적이면서 현대적이다.
